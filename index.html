cursor.com →

구글 계정으로 회원가입

Cursor 실행 후 구글 아이디로 회원가입을 진행합니다. Pro trial이 뜨면 Skip for now를 클릭하세요.

03

로그인 후 원하는 테마를 선택합니다. Agent 옵션에서 continue 클릭 → 언어를 한국어로 변경합니다. (추후 변경 가능)

04
프로젝트 폴더 만들기

바탕화면에 새 폴더 생성 (폴더명은 영어로). Cursor에서 Open Project를 클릭하여 생성한 폴더를 선택합니다.

폴더명 영어로 작성!
05
한국어 언어팩 설치

왼쪽 위 Extension 메뉴 → korea 검색 → Korean Language Pack 설치 → Change Language and Restart 클릭

06
Live Server 설치

왼쪽 위 Extension 메뉴 → Live Server 검색 → 설치

실시간 미리보기 가능해짐

⌨️ 간단한 단축키

Ctrl + L 오른쪽 채팅창 열기 → Agent 선택
Ctrl + K 코드 수정 인라인 요청
Ctrl + Enter 변경 사항 Accept (확인)
F12 개발자 도구 열기 (Chrome)
Agent 설정 팁: Agent 옆 Auto를 선택하여 Auto를 해제하고, 원하는 모델(예: claude-4-sonnet)을 선택하세요.

💡 더 직접적으로 수정 요청하는 방법

방법 1 — 개발자 도구 활용

  • F12 또는 오른쪽 점3개 → 도구 더보기 → 개발자 도구
  • 수정할 영역에 마우스를 가져다 클릭
  • 해당 부분의 HTML 요소가 선택됨
  • Ctrl + Shift + C 로 요소 복사
  • 채팅창에 붙여넣고 수정 요청

방법 2 — 인라인 수정

  • 수정할 코드 부분을 드래그 선택
  • Ctrl + K 단축키 입력
  • 원하는 변경 사항을 자연어로 입력
  • 코드를 계속해서 자연어로 수정 가능
2

첫 번째 웹사이트 만들기 (투두리스트)

목표: HTML, CSS, JS만을 이용하여 투두리스트 웹페이지를 만들기. 각 파일은 분리하여 작성합니다.
투두리스트 웹사이트 할 일 추가/제거/수정 달력으로 보기 로컬스토리지 저장
01
Cursor에서 채팅창 열기

Ctrl + L → Agent 선택 → Auto 해제 → 원하는 모델 선택 (claude-4-sonnet 추천)

02
자연어로 웹사이트 구성 요청

아래와 같이 원하는 기능을 구체적으로 작성하여 붙여넣으세요:

HTML, CSS, JS만을 이용해서 내가 원하는 웹페이지를 만들고 싶어.

- 투두리스트라는 제목의 웹사이트
- 내가 할 일을 추가 제거 수정 할 수 있음
- 달력으로도 볼 수 있음
- 추가된 할일들은 로컬스토리지에 저장
- 파스텔 톤의 현대적인 디자인
03
생성된 코드 확인 및 Accept

코드와 파일들이 만들어지면 Accept File 버튼을 클릭하거나 Ctrl + Enter를 눌러 확인합니다.

04
Live Server로 미리보기

index.html 파일에서 Live Server 실행 → 브라우저에서 실시간으로 확인

수정사항을 자연어로 계속 요청하며 완성
예시 코드 참고: gw1.kr/호똑쌤예시코드에서 예시 코드를 확인할 수 있습니다.
